# tgbot_remote_controller

This bot is for controlling your laptop from your phone. 
It allows you to switch music tracks, pause videos, and more. 
You won't need to get up from the couch to pause a video!

### Run project

1) ```pip install -r requirements.txt```

2) Create tg bot (on next step you will use API token bot)

3) run prepare.py

4) run main.py
